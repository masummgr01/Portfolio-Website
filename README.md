# Portfolio Website

Hello everyone! Welcome to my portfolio website. Thanks for visiting!

**Live at:** https://masummgr01.github.io/Portfolio-Website/

## Setup Instructions

1. Add your profile photo:
   - Save your profile photo as `profile.jpg` in the `img` folder
   - The image should be square (recommended: 400x400px or larger)
   - Supported formats: JPG, PNG, or WebP

2. The website will automatically:
   - Fetch your GitHub projects from masummgr01
   - Display GitHub contributor stats
   - Show your repositories (excluding Portfolio-Website)

## Features

- Modern, responsive design
- Animated typing effect
- GitHub integration
- Project showcase
- Contact section
